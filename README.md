# E-Commerce Sales & Customer Analysis

## Project Overview
This project analyzes an e-commerce sales dataset using Excel and Power BI to identify revenue trends, customer behavior patterns, and product performance insights. The data was cleaned and transformed before visualization.

## Objective
- Analyze overall sales performance
- Identify top customers and best-selling products
- Track monthly and category-wise revenue trends
- Build an interactive dashboard for KPI monitoring

## Tools Used
- Advanced Excel (Data Cleaning, Pivot Tables, Data Transformation)
- Power BI (Data Modeling, DAX, Interactive Dashboard)

## Data Preparation
- Removed duplicates and handled missing values
- Standardized date and numeric formats
- Created calculated columns for better analysis

## Key Insights
- A small percentage of customers contributed to a large share of revenue
- Sales showed noticeable seasonal variation
- Certain product categories generated higher profit margins

## Outcome
Developed an interactive Power BI dashboard that helps track KPIs such as Total Revenue, Monthly Sales Trend, Product Performance, and Customer Contribution for better business decision-making.
